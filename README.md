# Practicing Kotlin
In this repo I have done various examples to get introduced to Kotlin.

Some basic topics covered:
- Arrays
- Ranges 
- Collection operators
- Conditionals
- Functions and Higher order functions
- Recursions
- Data types
- Exception handling
- Lists and maps
- Inheritance and interface
- Null safety
